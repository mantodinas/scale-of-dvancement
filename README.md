# Scale of Advancement

A high-fidelity, multi-page web exploration of the **Kardashev Scale**, designed to track and visualize the technological evolution of cosmic civilizations.

## 🚀 Project Overview
**Scale of Advancement** is a static web experience that categorizes civilizations based on their energy consumption—from our current Type 0.73 status to the mastery of entire galaxies.

### Key Features:
* **Immersive Visuals:** High-resolution atmospheric backgrounds generated via Midjourney, optimized in **WebP** format for lightning-fast load times.
* **Responsive Design:** Fully fluid layout using CSS Grid and Flexbox, ensuring a premium experience on desktop, tablet, and mobile.
* **Glassmorphism UI:** Modern, semi-transparent interface elements with backdrop-blur effects for high readability against cosmic imagery.
* **Progress Tracking:** Custom CSS-animated progress bar visualizing humanity's journey toward a Type I civilization.

## 🛠️ Built With
* **HTML5:** Semantic structure for better SEO and accessibility.
* **CSS3:** Custom variables, Glassmorphism effects, and responsive `@media` queries.
* **Midjourney:** AI-driven art for the "Identity" logo and 16:9 cinematic backgrounds.
